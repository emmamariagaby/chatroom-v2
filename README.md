# chatroom-v2

This project is a refactor of project: https://github.com/emmamariagaby/chat-app

This project is a chat application who handles chatting clients. Every client can create a new room and join theirs and others. When client creates a room they can choose to give the room a password, a room with a password can only be join by writing correct password and it handles validation. All client rooms are listed in a sidebar where the clients can see what rooms are open and rooms disappears when they're empty from clients.

## Prerequisites

You need to have the following installed

- node.js
- npm

## How to run the program

Run in root folder of project:

1. npm install
2. npm start

## Running on multiple computers

If you want to run on multiple computers you have to specify the local ip address of the host computer in your browser console before joining the lobby.

Example:
host = "http://192.168.1.01:3000"
